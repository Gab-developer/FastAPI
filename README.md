# FastAPI-Simples

Uma API simples com **FastAPI** para aprendizao.

---

## Funcionalidades
- ✅ Endpoints REST básicos
- ✅ Documentação automática via Swagger UI (`/docs`)
- ✅ Fácil de entender e expandir

---

## 🔹 Endpoints
- `GET /` – Mostra todo banco de dados.
- `POST /create_user` – Cria um novo usuário.
- `PUT /update_user/{user_id}` – Atualiza um usuário existente.
- `DELETE /delete_user/{user_id}` - Deleta um usuário existente.
